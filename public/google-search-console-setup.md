# MIDASBUY GOOGLE SEARCH CONSOLE SETUP GUIDE

## 🚀 Quick Setup Instructions

### 1. GOOGLE SEARCH CONSOLE ACCESS

**Main URL:** https://search.google.com/search-console

### 2. ADD YOUR PROPERTY

- Click "Add Property"
- Select "URL prefix"
- Enter: `https://www.middasbuy.com`
- Verify ownership using HTML file method or DNS

### 3. SUBMIT SITEMAP

**Sitemap URL to submit:** `https://www.middasbuy.com/sitemap.xml`

**Steps:**

1. Go to Search Console Dashboard
2. Click "Sitemaps" in left sidebar
3. Enter: `sitemap.xml`
4. Click "SUBMIT"

### 4. ROBOTS.TXT VERIFICATION

**Check robots.txt:** https://www.middasbuy.com/robots.txt

- Should be automatically accessible
- Contains sitemap reference
